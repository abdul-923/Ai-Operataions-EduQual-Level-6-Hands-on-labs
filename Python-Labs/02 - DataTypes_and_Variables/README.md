# Lab 02 - Data Types and Variables

## Objective

The objective of this lab was to learn about Python data types, create variables, and display their values along with their data types.

## What I Practiced

- Creating variables in Python
- Using String, Integer, Boolean and Float data types
- Printing variable values
- Checking variable data types using the type() function
- Running Python files from the Linux terminal

## Code

- name = "Jhon"    # String
- age = 22         # Integer
- is_student = True    # Boolean
- height = 175.4   # Float
-
- print("Name:", name, "Type:", type(name))
- print("Age:", age, "Type:", type(age))
- print("Is Student:", is_student, "Type:", type(is_student))
- print("Height:", height, "Type:", type(height))

## Commands Used

- nano variables_lab.py
- python3 variables.py
- python3 variables_lab.py
- cat variables_lab.py

## Output

- Name: Jhon   Type: <class 'str'>
- Age: 22   Type: <class 'int'>
- Is Student: True   Type: <class 'bool'>
- Height: 175.4   Type: <class 'float'>

## What I Learned

In this lab, I learned that every value in Python has a data type. I practiced creating variables of different types and used the type() function to check their data types. I also learned the difference between String, Integer, Boolean, and Float values.
