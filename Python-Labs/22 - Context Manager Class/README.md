

# Lab 22 - Python Context Managers

## Objective

Learn how to use Python Context Managers with the 'with' statement to automatically manage resources such as files.

## What I Learned

* Creating a custom context manager class
* Using the '__enter__()' method
* Using the '__exit__()' method
* Working with the 'with' statement
* Managing files automatically
* Opening and closing files without calling 'close()'
* Writing data to a text file

## Files

* con_man.py
* sample.txt

## How to Run

python3 con_man.py

## Expected Output

Entering

Running within the context block.

Exiting

Contents of sample.txt

Hello, World!
