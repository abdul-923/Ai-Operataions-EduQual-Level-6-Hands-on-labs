## Concepts Learned

### Defining a Function

A function is a reusable block of code that performs a specific task.

Example:

def greet(name):


### Calling a Function

Calls the function and passes a value to the parameter.

Examples:

greet("Alice")

greet("Huda")

greet("Amna")


### Default Parameter

A default value is used if no argument is provided.

Example:

def greet(name="Guest"):


### Keyword Argument

Passes the value using the parameter name.

Example:

greet(name="Diana")
