# Lab 04 - If Else Lab

## Objective

The objective of this lab was to understand how decision-making works in Python using `if`, `elif`, and `else` statements.

## What I Practiced

- Taking integer input from the user
- Using `if`, `elif`, and `else` conditions
- Comparing values using comparison operators
- Displaying different outputs based on user input
- Running Python programs from the Linux terminal

## Code

- score = int(input("Enter Your Score"))
-
- if score >= 90:
-     print("Grade: A")
- elif score >= 80:
-     print("Grade: B")
- elif score >= 60:
-     print("Grade: C")
- else:
-     print("Grade: D")

## Commands Used

- nano if_else.py
- python3 if_else.py

## Sample Output

- Enter Your Score: 74
- Grade: C

## What I Learned

In this lab, I learned how Python makes decisions using conditional statements. I practiced checking multiple conditions with `if`, `elif`, and `else` to determine the correct grade based on the user's score. I also learned that Python executes only the first condition that evaluates to `True`.
