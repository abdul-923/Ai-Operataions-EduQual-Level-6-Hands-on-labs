

# Lab 23 - Basic SQLite Usage

## Objective

Learn how to use SQLite with Python by creating a database, creating a table, inserting records, retrieving data, and displaying the results.

## What I Learned

* Importing the built-in sqlite3 module
* Creating a connection to an SQLite database
* Creating a table using SQL
* Inserting multiple records with executemany()
* Saving changes using commit()
* Retrieving records using the SELECT statement
* Reading query results with fetchall()
* Closing the database connection using close()
