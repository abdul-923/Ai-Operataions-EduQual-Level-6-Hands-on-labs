## Concepts Learned

### Tuple

* Stores multiple values.
* Uses parentheses ().
* Ordered collection.
* Cannot be modified after creation (immutable).

Example:

names = ("ali", "ahmad", "java", "pythotin")



### Set

* Stores multiple unique values.
* Uses curly braces {}.
* Unordered collection.
* Duplicate values are automatically removed.

Example:

car = {"vigen", "bugati", "lambo"}

## Error Faced

### NameError

Problem:

names = (ali, ahmad, java, pythotin)

Reason:

The values were written without quotation marks, so Python treated them as variables instead of text.

Solution:

names = ("ali", "ahmad", "java", "pythotin")
