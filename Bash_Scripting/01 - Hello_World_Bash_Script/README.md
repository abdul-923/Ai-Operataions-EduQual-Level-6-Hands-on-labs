

# 01 - Hello World Bash Script

## Overview

This lab introduces the basics of Bash scripting. In this lab, I created my first Bash script, made it executable, and executed it from the terminal.

## Files

* hello.sh – Bash script created in this lab.
* README.md – Documentation for this lab.

## What I Practiced

* Creating a Bash script.
* Writing comments in Bash.
* Using the shebang (#!/bin/bash).
* Printing text using the echo command.
* Making a script executable.
* Running a Bash script from the terminal.
* Viewing file permissions.

## Commands Used

* nano hello.sh – Create or edit the Bash script.
* chmod +x hello.sh – Give execute permission to the script.
* ls -l – View file permissions.
* ./hello.sh – Execute the Bash script.

## Output

The script prints the following messages:

* Hello World!
* This is my first Hands on Lab.
* Thanks.

## What I Learned

* Every Bash script usually starts with #!/bin/bash.
* The echo command is used to display text on the terminal.
* A script must have execute permission before it can be run.
* The command ./hello.sh runs the script from the current directory.
* The ls -l command shows file permissions.
