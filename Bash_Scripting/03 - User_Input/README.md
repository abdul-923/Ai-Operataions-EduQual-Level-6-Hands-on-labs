# 03 - User Input

## Overview

This lab demonstrates how to take input from the user in a Bash script using the `read` command and display the entered value using a variable.

## Files

* input.sh
* README.md
* Screenshot 1 – Bash script (source code)
* Screenshot 2 – Terminal output

## What I Practiced

* Displaying a message using 'echo'
* Reading user input with the 'read' command
* Storing user input in a variable
* Printing the variable using '$'
* Running a Bash script from the terminal

## Commands Used

* nano input.sh
* bash input.sh

## Output


## Learning Outcome

In this lab, I learned how to interact with the user by accepting input during program execution. I understood how the `read` command stores the user's input in a variable and how that variable can be displayed later using `echo`. This is a fundamental concept for creating interactive Bash scripts.
