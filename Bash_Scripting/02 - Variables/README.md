
#02 - Variables

## Overview

This lab introduces variables in Bash scripting. It demonstrates how to create variables, store text values, use variables with the `echo` command, and display dynamic output in the terminal.

## Files

* variables.sh - Bash script created in this lab.
* README.md - Documentation for this lab.
* Screenshot From 2026-07-20 15-43-xx.png - Bash script source code.
* Screenshot From 2026-07-20 15-43-xx.png - Terminal output after executing the script.

## What I Practiced

* Creating variables in Bash.
* Storing text values inside variables.
* Displaying variable values using the echo command.
* Using the '$' symbol to access variables.
* Executing a Bash script from the terminal.

## Commands Used

* nano variables.sh - Create or edit the Bash script.
* chmod +x variables.sh - Make the script executable.
* ./variables.sh - Run the Bash script.

## Output

The script prints a greeting using variables and displays another text message.

Example output:

Hello Anna! Alice

Whats Your intentions today

## Learning Outcome

After completing this lab, I learned how to create and use variables in Bash scripts and print their values using the 'echo' command. I also practiced editing and executing Bash scripts from the Linux terminal.
